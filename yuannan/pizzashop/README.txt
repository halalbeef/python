Read about.txt first

Files here  will be deleted after use,
Most of the scripts are testers and will be loaded into the junks folder in the /python/junks folder after usage
Feel free to steal the countdown script if you need it

The files needed to run are "start.py" and ALL of the files in /etc
The rest will be generated as it run.

NOTE: requires python 3.X+

Linux, Mac and most UNIX based system:
  Start a terminal
  cd {path of file}
  python {name of file}
  Order will  be printed into the same folder as it's ran in
  
Windows:
  Open the file via IDLE
  or double click, if it runs great, if not try one of the python installers at http://legacy.python.org/download/
  or use Cygwin
